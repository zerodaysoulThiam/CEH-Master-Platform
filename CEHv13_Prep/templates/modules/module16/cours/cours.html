1. Wireless Concepts

Fondamentaux Wi-Fi

  • SSID: Identifiant du réseau (Service Set Identifier)
  • BSSID: Adresse MAC du point d'accès
  • Canaux: Bandes de fréquence (2.4GHz, 5GHz, 6GHz)
  • Modes: Infrastructure vs Ad-hoc
  • Normes: 802.11a/b/g/n/ac/ax (Wi-Fi 6/6E)
Composants clés
  • Points d'accès (AP)
  • Cartes réseau sans fil (NIC)
  • Contrôleurs Wi-Fi
  • Antennes (omnidirectionnelles/directionnelles)
  • Client sans fil

Chiffres clés

  • 87% des foyers utilisent le Wi-Fi
  • 62% des entreprises ont subi des attaques Wi-Fi
  • 74% des réseaux publics sont vulnérables
  • Débit théorique Wi-Fi 6E: 10 Gbps

2. Wireless Encryption

Protocoles de sécurité

Protocole Sécurité Vulnérabilités
WEP Très faible IV collisions, RC4 faible
WPA Faible TKIP vulnérable
WPA2 Moyenne KRACK attack, PMKID
WPA3 Forte Dragonfly attack (théorique)

Mécanismes d'authentification

  • PSK: Pre-Shared Key (clé partagée)
  • EAP: Extensible Authentication Protocol
  • RADIUS: Authentification centralisée
  • Captive Portal: Authentification web

Attaque KRACK (Key Reinstallation Attack)

Exploite la réinstallation du handshake 4-way dans WPA2

aireplay-ng --deauth 100 -a [BSSID] [interface]

3. Wireless Threats

Types d'attaques

  • Rogue AP: Point d'accès non autorisé
  • Evil Twin: Clone d'AP légitime
  • Deauth Attack: Désauthentification clients
  • MITM: Man-in-the-Middle via ARP spoofing
  • Packet Sniffing: Écoute du trafic non chiffré
  • WPS Attacks: Exploitation de PIN WPS

Impact des attaques

  • Interception de données sensibles
  • Usurpation d'identité
  • Déni de service
  • Propagation de malware
  • Accès au réseau interne

Exemple: Evil Twin

  1. Détecter le SSID cible
  2. Créer un AP avec même SSID
  3. Forcer les déconnexions de l'AP légitime
  4. Capturer les identifiants via captive portal

4. Wireless Hacking Methodology

Processus d'attaque

  1. Reconnaissance: Détection des réseaux
  2. Scanning: Analyse des AP et clients
  3. Fingerprinting: Identification du matériel
  4. Attaque: Exploitation des vulnérabilités
  5. Accès: Connexion au réseau
  6. Post-exploitation: Pivotage et persistance

Techniques de cracking

  • Dictionnaire: Utilisation de wordlists
  • Brute-force: Essai systématique
  • Rainbow Tables: Pré-calcul de hachages
  • GPU Acceleration: Hashcat avec GPU

Workflow WPA2

  1. Mettre l'interface en mode monitor
  2. Capturer le handshake 4-way
  3. Lancer une attaque de désauthentification
  4. Extraire le hash avec cap2hccapx
  5. Cracker avec hashcat/hcxdumptool

Commandes essentielles

airmon-ng start wlan0
airodump-ng wlan0mon
aireplay-ng --deauth 10 -a [BSSID] wlan0mon
aircrack-ng -w wordlist.txt capture.cap

6. Bluetooth Hacking

Technologies Bluetooth

  • Bluetooth Classic (BR/EDR)
  • Bluetooth Low Energy (BLE)
  • Bluetooth Mesh

Attaques courantes

  • Bluejacking: Envoi de messages non sollicités
  • Bluesnarfing: Vol de données
  • Blueborne: Exploitation sans appairage
  • BIAS: Bluetooth Impersonation Attack

Outils Bluetooth

  • hcitool: Contrôle des interfaces
  • bluelog: Détection des appareils
  • Bluetoothctl: CLI pour Bluetooth
  • BrakTooth: Exploit framework
  • GATTacker: Attaques MITM BLE

Exemple: Attaque BLE

gatttool -b [MAC] --char-write-req -a 0x002a -n 01
bettercap -iface hci0

7. Wireless Attack Countermeasures

Bonnes pratiques

  • Utiliser WPA3 ou WPA2 avec AES
  • Désactiver WPS
  • Changer les identifiants par défaut
  • Mettre à jour le firmware régulièrement
  • Utiliser un SSID unique
  • Configurer un VLAN invité

Configuration sécurisée

  • Désactiver l'administration à distance
  • Activer le filtrage MAC (limité)
  • Réduire la puissance du signal
  • Configurer le taux de renouvellement des clés

Politiques de sécurité

  • Authentification 802.1X/EAP
  • Certificats numériques pour les appareils
  • Segmentation réseau stricte
  • Surveillance continue du spectre
  • Tests d'intrusion réguliers

Contre-mesures Bluetooth

  • Désactiver Bluetooth si inutilisé
  • Utiliser le mode "non-découvrable"
  • Rejeter les appairages non sollicités
  • Mettre à jour les pilotes Bluetooth

8. Wireless Security Tools

Solutions professionnelles

  • WIPS: Wireless Intrusion Prevention Systems
  • NAC: Network Access Control
  • SIEM: Security Information and Event Management
  • Scanners RF: Détection des interférences

Outils open-source

  • Wireshark (analyse de trafic)
  • Kismet (détection rogue AP)
  • OpenWIPS-ng (système WIPS)
  • Snort (détection d'intrusion)

Méthodes de surveillance

  • Scans périodiques des réseaux
  • Détection des anomalies de trafic
  • Surveillance des canaux radio
  • Analyse des logs des points d'accès
  • Tests de pénétration automatisés

Audit de sécurité

  1. Cartographie du réseau sans fil
  2. Analyse des configurations
  3. Tests de résistance aux attaques
  4. Vérification de la couverture radio
  5. Rapport de vulnérabilités
